Voice Platform

Detail

An application created with the purpose of providing MC's voice for content creators on social networks.

Some creators have really good and quality content, but they can't convey all the emotions to the audience. So Voice Platform was born.

This application was built by 5 members in the process of studying Software Architecture and Design at FPT University. In which, I and @Ushio are in charge of providing API.

The project development team includes Le Huy, Le Phuc, Si Trong, Nguyen Bach and Tan Tai. This project has almost reached the maximum score with 9/10 graded by lecturer PhuongLHK at FPT University. The following are the technologies used in this project.

    • Flutter

    • SQL Server 2019

    • Firebase Storage
    • Firebase Authorize
    • Firebase Hosting
    • Sendgrid
    • Azure Devops
